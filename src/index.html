Wordle Solver Learning Demo

Open the console to see the output.

The solver is controlled via `app.js` within `const runSolver`. `solver.findBestWord()` returns the current best word to solve the Wordle.

  1. Grey letters (not present in the word) can be added with solver.addGreyLetter(char c).
  2. Orange letters (present but at wrong position) can be added with solver.addOrangeLetter(int position, char c), where position 0 corresponds to the 1st letter and position 4 to the 5th letter.
  3. Green letters (correct position) can be added with solver.addGreenLetter(int position, char c), using the same position numbering as orange letters.
